| Human CMS 2004 특징 |
 
▶CMS 사업 참여 사례 :
한국농림수산정보센터, 한국철도기술연구원, iMBC, 서울시청, 해양수산부, 부천시청
▶Human CMS 2004를 이용한 구축 공급 사례 :
마포구청, 한국표준협회(홈페이지, Q-NET, KSSN-NET), 한국국제전시장, 중앙선거관리위원회, 법제처

사이트메뉴 관리와 콘텐츠의 분류 체계를 별도 관리
콘텐츠 분류도에서 각 메뉴와 관련있는 콘텐츠만을 설정할 수 있는 기능(연관 콘텐츠)
연관 콘텐츠에서 실질적으로 퍼블리싱 대상이 되는 콘텐츠만을 목록화하여 관리하는 기능 (출판대상 콘텐츠)

사용자별, 그룹별 권한부여로 인하여 효율적인 사이트 관리업무 기능 강화
사이트 메뉴 뿐아니라 콘텐츠 별 권한관리로 인하여 유연한 설정이 가능

사용자별, 카테고리별, 템플릿별, 콘텐츠별, 퍼블리싱별, 기간별 로그를 통해서
CMS 사용 현황을 관리할 수 있는 기능 강화
날짜나 작업자 등 옵션에 따른 로그만을 조회 가능

자사 순수 개발로 고객사에 맞는 기능 추가 등 커스터마이징 용이
신속, 정확하게 고객 대응 가능
