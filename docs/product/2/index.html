| Human CMS 2004 기능 |
 

웹기반 사용자 인터페이스로 시간과 장소에 구애없이 콘텐츠 관리 가능
비주얼 에디터를 내장하여 HTML 콘텐츠 편집이 용이
별도의 클라이언트 설치작업 없이 웹 브라우저만으로 관리 가능

웹사이트 단위나 사이트 메뉴 단위로 배포 가능
다양한 조건의 배포 즉, 예약, 자동 스케줄링, 즉시 배포 가능

Java를 기반으로 이식성이 뛰어남
기존 시스템과의 연동을 위해 API 등 지원
모듈별 컴포넌트 베이스로 유지보수 및 확장성 용이

사용자 인터페이스의 다국어 지원
다국어 콘텐츠 관리 지원

디자인과 분리된 콘텐츠 자체만의 저장관리로 타 시스템과 콘텐츠 호환이 용이

한번의 콘텐츠 등록으로 웹서버 뿐만 아니라 PDA, WAP으로 배포 가능
각 장치의 고유 템플릿 지원
＊ 체계적인 콘텐츠 라이프사이클 관리 기능으로 CMS가 가지는 기본 기능을 모두 지원합니다.
