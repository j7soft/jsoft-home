| User Menus |
  Human CMS 2004 user interface consists of the seven major menus.

» Manage the access control for the organization and users

» Categorize, register, modify, and disuse contents

» Manage the web site menus using hierarchical folders

» Manage the commonly used components such as template codes and custom fields

» Manage the multiple web servers

» Manage the automatic publish conditions and session monitoring

» Manage logs by users, categories, templates, contents, publishing, and periods
