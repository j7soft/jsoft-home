| Basic Functions |
  Human CMS 2004 provides all the important CMS functions as follows.

» Can access the system anytime anywhere
» Easy to use WYSWYG HTML editor is included
» No client application set up (Needs only web browser)

» Can publish the contents for the whole web site or the site menu
» Can set various publishing options
(reserved, scheduled, and immediate publishing)

» Portable because of Java based system
» Support existing backend systems through the API
» Extendable because of modular component based design

» Support multi-lingual user interface
» Support multi-lingual contents

» Easy to port contents by managing them separately from the design templates

» Deploy the same contents for the PDA or the WAP as well as the Web environments without additional efforts
